# Lux Client

Lux Client is an iOS remote desktop client built using Swift and `xtool` (a cross-platform Swift package build tool). 

## Prerequisites
- A connected Apple device (for testing and deployment)
- [xtool](https://xtool.sh) installed 

## Building and Running
To build and deploy the application to your connected iOS device, simply run:
```bash
xtool dev
```
